The BackupsRunning table contains information about backups that are currently running.

Column Descriptions
sessionId – unique identifier of the Plesk user’s session
param – parameter keyword
val – value which corresponds to the parameter keyword

Table Structure

CREATE TABLE `BackupsRunning` (
 `sessionId` varbinary(250) NOT NULL default '                                                                                                                                                                                                                                                          ',
 `param` varchar(250) character set ascii NOT NULL default '',
 `val` varbinary(255) default NULL,
 PRIMARY KEY  (`sessionId`,`param`)
) ENGINE=InnoDB DEFAULT CHARSET=latin1
