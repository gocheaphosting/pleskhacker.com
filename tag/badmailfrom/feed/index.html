The badmailfrom table contains the domains which are blacklisted server-wide.

Column Descriptions
id – primary key that is automatically incremented
domain – domain name to add to the blacklist

Table Structure

CREATE TABLE `badmailfrom` (
 `id` int(10) unsigned NOT NULL auto_increment,
 `domain` varchar(255) character set utf8 NOT NULL default '',
 PRIMARY KEY  (`id`),
 KEY `domain` (`domain`)
) ENGINE=InnoDB AUTO_INCREMENT=2 DEFAULT CHARSET=latin1
