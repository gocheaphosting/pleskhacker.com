The mail_redir table contains the destination e-mail address for mail redirects.

Column Descriptions
id – primary key that is automatically incremented
mn_id – links to id in the mail table
address – destination e-mail address

Table Structure

CREATE TABLE `mail_redir` (
 `id` int(10) unsigned NOT NULL auto_increment,
 `mn_id` int(10) unsigned NOT NULL default '0',
 `address` varchar(245) character set utf8 NOT NULL default '',
 PRIMARY KEY  (`id`)
) ENGINE=InnoDB AUTO_INCREMENT=11 DEFAULT CHARSET=latin1
