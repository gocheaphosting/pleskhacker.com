The cl_param table helps Plesk link up certain client parameters with the client. These parameter can include the look of their login screen, how they move between screens in Plesk, and their preferred domain templates.

Column Descriptions
cl_id – links to id in clients table
param – text keyword for the parameter
password – value which corresponds to the parameter keyword

Table Structure

CREATE TABLE `cl_param` (
 `cl_id` int(10) unsigned NOT NULL default '0',
 `param` varchar(245) character set ascii NOT NULL default '',
 `val` varbinary(255) default NULL,
 PRIMARY KEY  (`cl_id`,`param`)
) ENGINE=InnoDB DEFAULT CHARSET=latin1
