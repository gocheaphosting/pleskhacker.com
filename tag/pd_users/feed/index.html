The pd_users table contains the users for the protected directories on the server.

Column Descriptions
id – primary key that is automatically incremented
login – the username for the user
account_id – links to id in accounts table
pd_id – links to id in protected_dirs table

Table Structure

CREATE TABLE `pd_users` (
 `id` int(10) unsigned NOT NULL auto_increment,
 `login` varchar(20) character set ascii collate ascii_bin NOT NULL default '',
 `account_id` int(10) unsigned NOT NULL default '0',
 `pd_id` int(10) unsigned NOT NULL default '0',
 PRIMARY KEY  (`id`),
 UNIQUE KEY `pd_id` (`pd_id`,`login`),
 KEY `account_id` (`account_id`)
) ENGINE=InnoDB AUTO_INCREMENT=31 DEFAULT CHARSET=latin1
